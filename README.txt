《小王子的亲子问答星球》部署说明

1. 准备文件：
   - index.html (网页主文件)
   - questions.json (问答数据文件)

2. 部署步骤：
   - 登录阿里云控制台，进入对象存储 OSS。
   - 创建或选择一个 Bucket。
   - 将 index.html 和 questions.json 上传到该 Bucket 的根目录。
   - 在 Bucket 配置中，找到“静态页面”设置，将“默认首页”设置为 index.html。
   - 确保文件读写权限设置为“公共读”。
   - 通过 OSS 提供的外网访问域名即可访问您的问答星球。

3. 注意事项：
   - 请确保两个文件在同一目录下，否则 fetch 加载数据会失败。
   - 建议在手机端访问以获得最佳亲子互动体验。
